# Images du site — assets/media

Toutes les images du site sont dans **`assets/media/`**. Ce sont des copies des fichiers nécessaires du **Revolut media kit** (dossier non versionné). **Ne poussez jamais** le dossier complet « Revolut media kit » sur le dépôt.

## Mise en ligne

Les meta `og:image` et `twitter:image` pointent vers :
`https://YOUR-DOMAIN.com/assets/media/og-image.png`

Remplacez `YOUR-DOMAIN.com` par votre domaine dans `index.html`. Déployez le dossier **`assets/media`** avec le site.
